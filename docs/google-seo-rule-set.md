# Google Search Console SEO Rule Set

Based on Google's official Search Console documentation and best practices for maximum search visibility.

## 🎯 **CORE GOOGLE SEO PRINCIPLES**

### **1. INDEXING & DISCOVERY RULES**
- **Sitemap Submission**: Always submit sitemaps through [Sitemaps report](https://search.google.com/search-console/sitemaps) - Google's preferred method
- **URL Inspection**: Use [URL Inspection tool](https://search.google.com/search-console?action=inspect) to test individual pages before launch
- **Page Indexing**: Monitor [Page indexing report](https://search.google.com/search-console/index) for coverage issues
- **Robots.txt Validation**: Use [robots.txt report](https://search.google.com/search-console/settings/robots-txt) to ensure proper crawling

### **2. PERFORMANCE OPTIMIZATION RULES**
- **Core Web Vitals**: Must pass [Core Web Vitals report](https://search.google.com/search-console/core-web-vitals) - Google's ranking factor
- **Performance Monitoring**: Track [Performance reports](https://search.google.com/search-console/performance/search-analytics) for clicks, impressions, CTR, and position
- **Mobile Performance**: Ensure mobile-first indexing compatibility
- **Page Speed**: Optimize for Google's page experience signals

### **3. CONTENT & STRUCTURE RULES**
- **Structured Data**: Use [Rich Results Test tool](https://search.google.com/test/rich-results) for validation
- **AMP Implementation**: Monitor [AMP report](https://search.google.com/search-console/amp) for mobile optimization
- **Video Content**: Track [Video indexing report](https://search.google.com/search-console/video-index) for video SEO
- **Canonical URLs**: Ensure proper canonicalization to avoid duplicate content

### **4. TECHNICAL SEO RULES**
- **HTTPS Enforcement**: Monitor [HTTPS report](https://search.google.com/search-console/security-issues) for security
- **Manual Actions**: Check [Manual actions report](https://search.google.com/search-console/manual-actions) regularly
- **Security Issues**: Address any [Security issues report](https://search.google.com/search-console/security-issues) immediately
- **Crawl Stats**: Monitor [Crawl stats](https://search.google.com/search-console/settings/crawl-stats) for crawl budget optimization

## 🔄 **GOOGLE-SPECIFIC SEO WORKFLOW**

### **DAILY CHECKS**
1. **Overview Page**: Check for manual actions or security issues
2. **Performance Spikes**: Look for dramatic dips in click count
3. **Coverage Errors**: Monitor for spikes in coverage errors

### **WEEKLY REVIEWS**
1. **Page Indexing**: Scan for drops in indexed counts
2. **Sitemap Status**: Verify sitemap processing success
3. **Core Web Vitals**: Check real-world performance metrics
4. **Rich Results**: Validate structured data implementation

### **MONTHLY AUDITS**
1. **Performance Analysis**: Review query strings and search visibility
2. **URL Inspection**: Test important pages for indexing issues
3. **Removals Tool**: Manage blocked URLs and search snippets
4. **Links Report**: Analyze internal and external link patterns

## 🔧 **GOOGLE SEARCH CONSOLE INTEGRATION RULES**

### **PROPERTY MANAGEMENT**
- **Domain vs URL-Prefix**: Use domain-level properties when possible for full crawl stats
- **Ownership Verification**: Maintain multiple verification methods
- **User Permissions**: Limit access to necessary team members only
- **Associations**: Link with Google Analytics, YouTube, Android apps

### **DATA EXPORT & ANALYSIS**
- **Bulk Data Export**: Use [Bulk data export](https://support.google.com/webmasters/answer/12917675) to BigQuery for advanced analysis
- **Performance Insights**: Leverage [Insights report](https://search.google.com/search-console/performance/insights) for quick overviews
- **Search Analytics**: Deep dive into [Performance reports](https://search.google.com/search-console/performance/search-analytics) for detailed metrics

## 📱 **GOOGLE-SPECIFIC CONTENT RULES**

### **DISCOVERY & NEWS OPTIMIZATION**
- **Discover Traffic**: Optimize for [Discover performance](https://search.google.com/search-console/performance/discover) if applicable
- **Google News**: Monitor [News performance](https://search.google.com/search-console/performance/google-news) for news sites
- **Trending Content**: Use insights to identify content opportunities

### **VIDEO & MULTIMEDIA SEO**
- **Video Indexing**: Ensure videos meet [video indexing criteria](https://developers.google.com/search/docs/appearance/video)
- **Watch Page Optimization**: Focus on video watch pages for indexing
- **Video Features**: Optimize for Google's video features in search

## 🚀 **GOOGLE RANKING FACTORS COMPLIANCE**

### **PAGE EXPERIENCE SIGNALS**
1. **Core Web Vitals**: LCP < 2.5s, FID < 100ms, CLS < 0.1
2. **Mobile-Friendly**: Responsive design and mobile optimization
3. **HTTPS Security**: SSL certificate and secure connections
4. **Safe Browsing**: No malware or phishing content

### **CONTENT QUALITY SIGNALS**
1. **E-A-T**: Expertise, Authoritativeness, Trustworthiness
2. **Freshness**: Regular content updates and lastmod dates
3. **Relevance**: Content matching user search intent
4. **Completeness**: Comprehensive coverage of topics

## 🕷️ **GOOGLE CRAWL OPTIMIZATION**

### **CRAWL BUDGET MANAGEMENT**
- **Crawl Stats Monitoring**: Track Google's crawl frequency and requests
- **Site Architecture**: Optimize internal linking structure
- **Duplicate Content**: Minimize with proper canonicals
- **Error Handling**: Fix 404s and server errors promptly

### **SITEMAP OPTIMIZATION**
- **Dynamic Generation**: Keep sitemaps current with content changes
- **Lastmod Dates**: Include accurate modification timestamps
- **Priority Settings**: Use priority tags for important pages
- **Change Frequency**: Set appropriate change frequency values

## 🚨 **GOOGLE-SPECIFIC MONITORING ALERTS**

### **CRITICAL ALERTS**
- Manual actions applied to site
- Security issues detected
- Significant drops in indexed pages
- Core Web Vitals failures

### **PERFORMANCE ALERTS**
- Dramatic dips in click count
- Spikes in coverage errors
- Sitemap processing failures
- Rich results validation errors

## 📊 **GOOGLE SEARCH CONSOLE REPORTING RULES**

### **REPORT PRIORITIZATION**
1. **High Priority**: Overview, Page Indexing, Manual Actions
2. **Medium Priority**: Performance, Core Web Vitals, Sitemaps
3. **Low Priority**: Links, AMP, Video Indexing (if applicable)

### **ACTION ITEMS WORKFLOW**
1. **Identify**: Use reports to identify issues
2. **Investigate**: Use URL Inspection for detailed analysis
3. **Fix**: Implement technical or content fixes
4. **Validate**: Re-test with Rich Results Test tool
5. **Monitor**: Track improvements in subsequent reports
